# 选择一个模式
 1.[纯官方hosts(部分可能被墙, 总体速度较快)](https://github.com/WeiLDavid/hosts/raw/yt/resources/yt_offical)<br>
 2.[纯公共服务器hosts(信息不安全, 速度慢, 但可以在低清画质即144p下流畅观看, 不会被墙)](https://github.com/WeiLDavid/hosts/raw/yt/yt_pure_3rd)<br>
